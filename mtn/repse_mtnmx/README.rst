REPSE Module
=================

* This module install and enabled all the tools to manage repse documentation

Installation
=============
* Install the module from the Odoo Apps menu.
* Ensure all dependencies are met.
* Register the leader and members for activity team Activities for REPSE on Settings / Technical / Discuss / Activity Teams

Configuration
=============

* Technical module has to be manually installed
* Only Repse Manager / User / Portal User groups has access to the data. The REPSE menu is exclusive for the Repse Managers.

Usage
=====

* No special instructions given
